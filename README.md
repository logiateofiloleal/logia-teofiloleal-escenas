# Logia Teófilo Leal N° 115 — Maqueta Visual Scroll-Driven

Repositorio de maqueta visual para la nueva experiencia web de la **Respetable Logia Teófilo Leal N° 115**, Oriente de Barquisimeto.

Este proyecto representa una propuesta visual experimental tipo **scroll-driven**, donde el visitante/profano recorre una experiencia simbólica e inmersiva antes de llegar al llamado principal: **“Tocar la puerta”**.

## Objetivo del proyecto

El objetivo de esta maqueta es explorar una página principal más moderna, ceremonial e institucional, alejándose de una web tradicional por secciones y acercándose a una experiencia visual continua.

La propuesta busca transmitir:

* solemnidad institucional;
* identidad masónica;
* recorrido simbólico;
* profundidad visual;
* narrativa de entrada;
* conexión con la memoria de Teófilo Leal;
* llamado final al proceso de aspirante.

## Estado actual

Esta versión funciona como **laboratorio visual** y base de revisión.

Incluye:

* preloader institucional;
* experiencia principal por escenas;
* navegación tipo menú hamburguesa;
* escena dedicada a Teófilo Leal;
* acceso a reseña histórica;
* escena “Tocar la puerta”;
* formulario de aspirante;
* login y panel administrativo;
* assets visuales de referencia;
* cierre reflexivo institucional.

## Estructura principal

```txt
assets/              Recursos visuales e imágenes
backend/             Servidor y lógica backend
css/                 Estilos principales y estilos de escenas
js/                  Scripts principales y lógica scroll-driven
pages/               Páginas internas
index.html           Página original/base del proyecto
index-escenas.html   Maqueta visual scroll-driven
package.json         Dependencias del proyecto
```
